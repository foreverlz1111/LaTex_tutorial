<b name="2">说明：</b>

`可以直接修改.tex信息内容，然后跳过以下内容`

自定义命令：

- 顶部的那个标题：
```
\mytitle{}
```
- 灰色的标题条：
```
\barline{}
```
- 一行的四个单元格填充：
```
\tuplefourline{}{}{}{}
```
- 一行的（左右分布）四个单元格填充：
```
\lrline{左}{左}{左}{右}
```
- 经验、项目的介绍文本：
```
\contentbox{}
```
- 定义的颜色：
```
grey-back % （标题）灰色
blue3-text %下面的项目名（淡蓝色）
```
